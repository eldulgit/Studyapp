# StudyApp 패키지 구성 분석

> APK 역분석 결과 | 패키지명: `com.example.studyapp`

\---

## 기술 스택

|항목|내용|
|-|-|
|언어|Kotlin|
|UI|Jetpack Compose + Material3|
|네비게이션|Navigation Compose|
|DI|Hilt (`AppModul`, `DatabaseModul`, `RepositoryModul`)|
|이미지 로딩|Coil (Compose용)|
|로컬 저장소|DataStore Preferences (Room 아님)|
|비동기|Coroutines + Flow|
|빌드|AGP 8.13.2|

\---

## 네비게이션 구조

### BottomNavItem (탭 4개)

```
Calendar  →  CalendarScreen (+ TimeTableScreen 서브탭)
Timer     →  TimerScreen
Todo      →  TodoScreen          ← 와이어프레임에 없는 화면
Setting   →  SettingScreen (SettingHeader)
```

### Setting 하위 라우트 (MainScreen에서 9개 라우트 확인)

```
setting\_goal          →  GoalSettingScreen
setting\_notification  →  NotificationSettingScreen
setting\_theme         →  ThemeSettingScreen
setting\_account       →  AccountSettingScreen
setting\_ai            →  AiProfileSettingScreen
```

## DataStore 키 (설정 저장)

```
daily\_goal\_minutes     # 하루 목표 학습 시간 (분)
weekly\_goal\_minutes    # 주 목표 학습 시간 (분)
monthly\_goal\_minutes   # 월 목표 학습 시간 (분)
darkThemeEnabled       # 다크모드 여부
study\_settings         # 기타 학습 설정
```

## 주요 발견사항 및 오타/이슈

|위치|문제|비고|
|-|-|-|
|`BottomNabBar.kt`|오타: `Nav` → `Nab`|리팩터링 시 수정 필요|
|`PrimaryButtom.kt`|오타: `Button` → `Buttom`|리팩터링 시 수정 필요|
|`AppModul.kt` 등|오타: `Module` → `Modul`|DI 모듈 3개 전체|
|`GenerateDummyValues.kt`|Stats가 더미 데이터 사용 중|실제 DB 연동 필요|
|`StudyRepository.kt`|학습 기록 repo 존재하나 Room 테이블 미확인|DataStore만 사용 중인 듯|
|`AiProfileSettingScreen.kt`|와이어프레임에 없는 AI 설정 화면|기능 범위 확인 필요|
|`TodoScreen.kt`|와이어프레임에 없는 Todo 화면|탭에 존재|
